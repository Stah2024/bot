# Репостер Telegram → VK (многоканальный)

Самый мощный репостер 2025 года:  
- Один пользователь → сколько угодно каналов  
- Один канал → сколько угодно групп ВК  
- Фото + видео + текст  
- Шифрование токенов  
- Подписка/оплата (в разработке)

## Быстрый старт (5 минут)

`bash
git clone https://github.com/Stah2024/bot.git
cd bot
cp .env.example .env
nano .env        # вставить BOT_TOKEN и CRYPTO_KEY
pip install -r requirements.txt
pm2 start main.py --name tg2vk --interpreter python3
pm2 save

.env пример
BOT_TOKEN=8246854614:AAE...
CRYPTO_KEY=g53q.................uVw=
